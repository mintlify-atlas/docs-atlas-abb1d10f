The Gutenberg plugin brings the latest block editor features to your WordPress site before they become part of WordPress Core. You can install it directly from the WordPress plugin repository.

## Prerequisites

Before installing Gutenberg, ensure you have:

- WordPress 5.0 or higher
- A compatible hosting environment with PHP 7.0+
- Administrative access to your WordPress site

## Installation Methods

<Tabs>
  <Tab title="Via WordPress Admin">
    <Steps>
      <Step title="Access the Plugins Page">
        Log in to your WordPress admin dashboard and navigate to **Plugins > Add New**.
      </Step>
      
      <Step title="Search for Gutenberg">
        In the search box, type "Gutenberg" and press Enter. The official Gutenberg plugin should appear at the top of the results.
      </Step>
      
      <Step title="Install the Plugin">
        Click the **Install Now** button next to the Gutenberg plugin.
      </Step>
      
      <Step title="Activate the Plugin">
        Once installation completes, click **Activate** to enable the plugin on your site.
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Manual Upload">
    <Steps>
      <Step title="Download the Plugin">
        Visit the [Gutenberg plugin page](https://wordpress.org/plugins/gutenberg/) on WordPress.org and click the **Download** button to get the latest version.
      </Step>
      
      <Step title="Upload to WordPress">
        In your WordPress admin, go to **Plugins > Add New > Upload Plugin**. Click **Choose File** and select the downloaded `gutenberg.zip` file.
      </Step>
      
      <Step title="Install and Activate">
        Click **Install Now**, then **Activate Plugin** once the upload completes.
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Via FTP">
    <Steps>
      <Step title="Download and Extract">
        Download the plugin from WordPress.org and extract the ZIP file on your computer.
      </Step>
      
      <Step title="Upload via FTP">
        Connect to your server via FTP and upload the extracted `gutenberg` folder to your `/wp-content/plugins/` directory.
      </Step>
      
      <Step title="Activate in WordPress">
        Log in to WordPress admin, navigate to **Plugins**, find Gutenberg in the list, and click **Activate**.
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Verify Installation

After activation, you can verify that Gutenberg is running:

1. Create or edit a post or page
2. You should see the block editor interface with the latest Gutenberg features
3. Check the Gutenberg version in **Plugins** to ensure you have the latest release

<Note>
The Gutenberg plugin receives frequent updates with new features and improvements. We recommend keeping it updated to access the latest block editor capabilities.
</Note>
